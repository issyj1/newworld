Isabelle Johnson Photography - Site Design

Web Development, Design, UX Design, Web Design.

Web coding, design and front end development project. Using HTML, CSS and Javascript.

Front page design featuring image grid plus toggle dropdown menu with fade effect. There is also a modern overlay drop down to show Contact and About sections with images still visible underneath.

The perspective shift effect visible as a motif on the website, as a hover effect on the testimonials image.

The perspective shift effect is also used on the image gallery, where the images shift in and out of the frame and the perspective tilts as they exit the main picture. This really brings the images to life on the screen.
